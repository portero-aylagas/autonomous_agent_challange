# Autonomous Agent Challenge

This repository contains the planning deliverables for the Ironhack `LAB | Autonomous Agent Challenge`.

The chosen concept is an enterprise-style `QA Intake Agent` for automotive infotainment touchscreen UI defects. The submission is intentionally planning-focused and does not include a runnable prototype.

## Repository Map

- `project_plan.md`: Main lab deliverable with the full project plan
- `lab_summary.md`: Short reflection of the project

## How to Use This Repository

Read `project_plan.md` for the full solution. `lab_summary.md` contains the required short reflection paragraph. `lab_source/` is included only as reference material and was not modified.
